I.   Повсякденні завдання:
	1. Знайти квиток.
	2. Замовити квиток.

II.  Аналітика та звітність:
    	1. Знайти найближчий рейс до певного міста.
	2. Визначити кількість вільних місць на певний маршрут.

III. Словники:
	1. Список рейсів
	2. Аеропорти



Користувач має можливість:

1. Вибрати певне місто. 
2. Вибрати потрібну дату рейсу.
3. Переглянути рейси.
4. Вибрати потрібний рейс.
5. Вибрати необхідну кількість місць.
6. Отримати квиток(інформацію про квиток) | Отримати інформацію про відсутність місць.
